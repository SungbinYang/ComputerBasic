## 다나와에 가면
[다나와](https://www.danawa.com/)라는 사이트를 가보면 용어 설명이 링크로 잘 설명되어 있다. 틈틈이 용어를 읽어두고 익숙하도록 노력하자.

### CPU
- Core 개수가 CPU에 중요한데 Core란 연산하는 주체이다.

### Thread
- 실행하는 단위로 1000개의 Thread에 Core가 6개라면 1000개의 Thread 알고라룰 6개의 Core가 번갈아 가면 수행한다.

### MainBoard
- PS/2, VGA, Type C, HDMI, USB 3.0, USB2.0등이 소켓처럼 장착되어 있는데 고급진 용어로 인터페이스라고 불린다. 인터페이스란 즉, 컴퓨터와 주변기기를 연결시켜주는 접점의 역할을 한다.
